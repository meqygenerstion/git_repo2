## 1 Лабораторная работа

Програмный код, который позволяет находить с помощью программы число кратное n^3


## Скомпилировать
```bash
gcc -Wall main.c -o main
```

## Запусить

```bash
./main
```

## Help-команда
```bash
Usage:
  ./main <length> <n>

Meaning:
  length  - размер вектора (>=1)
  n       - целое >=1; ищем элементы, кратные n^3
```
